1. Visão Geral

O FlatPatternHighlight é uma ferramenta gratuita que roda dentro do Siemens NX 2512 e ajuda você a analisar automaticamente o flat pattern (padrão plano) de peças de chapa dobrada (Sheet Metal).

🔴 O problema que o plugin resolve

Quando você cria um flat pattern de uma peça de Sheet Metal no NX, o sistema gera dezenas de curvas e linhas. O problema é que:

  • O comando GetExteriorCurves retorna 74+ curvas — mas a maioria são de recortes, furos e alívios, não o contorno externo real.
  • Você precisa saber a distância de cada dobra até a borda da peça, mas fazer isso manualmente é tedioso e propenso a erro.
  • Identificar visualmente qual lado de cada dobra aponta para a borda externa é trabalhoso em peças com muitas dobras.

🟢 Como funciona

O plugin executa 3 etapas automáticas em sequência:

EtapaO que fazResultado
1. Perímetro Externo Filtra as curvas externas brutas e mantém apenas o contorno verdadeiro da peça, eliminando recortes e furos. Lista limpa de curvas do perímetro externo, destacadas na tela.
2. Linhas de Centro Localiza todas as linhas de centro de dobra (tanto as de dobra para cima quanto para baixo). Lista completa de linhas de centro, destacadas na tela.
3. Análise de Proximidade Para cada dobra, calcula a distância até a curva de perímetro mais próxima de cada lado. Gera um relatório completo e, se possível, cria cotas PMI em cadeia. Log detalhado + cotas PMI opcionais.
⏱ Tempo de execução típico: Entre 1 e 5 segundos, dependendo da complexidade da peça.

2. Instalação

📋 Requisitos

ItemExigência
Siemens NXVersão 2512 ou superior (licença Sheet Metal obrigatória)
Licença PMI/GD&TOpcional — necessária apenas para gerar cotas PMI automáticas
Sistema OperacionalWindows 10 ou 11 (64 bits)
PermissãoAcesso de escrita ao diretório startup do NX

📂 Copiar os arquivos

A instalação é simples: você só precisa copiar 3 arquivos para a pasta startup do NX.

Localize a pasta correta

O NX procura plugins na pasta:

%UGII_USER_DIR%\startup\

Geralmente este caminho é algo como:

C:\Users\SeuNome\AppData\Roaming\Siemens\NX2512\startup\

Se a pasta startup não existir, crie-a.

Copie os 3 arquivos

Copie estes arquivos para a pasta startup\:

ArquivoFunção
FlatPatternHighlight.dllO plugin em si (o programa)
FlatPatternHighlight.menRegistra o plugin no menu do NX
FlatPatternHighlight.rtbAdiciona um botão na faixa de opções (ribbon)
Importante: Os três arquivos devem estar na mesma pasta.
Reinicie o NX

Feche e abra o NX novamente. O plugin será carregado automaticamente.

✅ Verificar a instalação

Após reiniciar o NX, você deverá ver:

  • Um novo item de menu chamado "Flat Pattern" antes do menu Help
  • Uma nova aba na faixa de opções chamada "Flat Pattern" com um botão "Highlight"
⚠ Não apareceu? Verifique se:
  • Os 3 arquivos estão na pasta startup correta
  • A DLL foi gerada para a versão correta do NX (2512)
  • O NX foi reiniciado completamente

3. Executando o Plugin

Há 3 formas de executar o plugin. Escolha a que for mais conveniente.

📌 Pelo Menu

Abra uma peça com flat pattern

Certifique-se de que a peça ativa já possui um flat pattern gerado (Insert → Sheet Metal → Flat Pattern).

Acesse o menu

Na barra de menus do NX, clique em "Flat Pattern" (que aparece antes do menu "Help").

Clique na opção

Selecione "Highlight Exterior Curves".

Aguarde a execução

O plugin processa as 3 etapas em alguns segundos. O resultado é escrito no Log File do NX.

🎯 Pela Ribbon (Faixa de Opções)

Vá para a aba "Flat Pattern"

Na faixa de opções do NX, localize a aba chamada "Flat Pattern".

Clique no botão

No grupo "Highlight", clique no botão "Highlight Exterior Curves".

⌨️ Pelo Ctrl+U (Executar NX Open)

Pressione Ctrl+U

Ou vá em File → Execute → NX Open.

Selecione a DLL

Navegue até a pasta onde está o arquivo FlatPatternHighlight.dll e selecione-o.

Clique em OK

O plugin será executado imediatamente.

Importante: O plugin só funciona se a peça ativa já tiver um flat pattern gerado. Se você receber a mensagem "No Flat Pattern feature found", primeiro crie o flat pattern usando Insert → Sheet Metal → Flat Pattern.

4. Entendendo o Resultado

📄 Onde encontrar o Log File

Após a execução, o plugin escreve um relatório completo no Log File do NX. Para visualizá-lo:

  1. Vá em Help → Log File (ou Ajuda → Arquivo de Log)
  2. O log é aberto no editor de texto do NX
  3. Role até o final para ver a seção começando com === FlatPatternHighlight Diagnostic Log ===
💡 Dica: Você pode copiar todo o conteúdo do log e colar em um editor de texto (Bloco de Notas, Excel, etc.) para análise posterior.

O log é dividido em seções que correspondem às 3 etapas. Veja um exemplo comentado:

👁️ O que os destaques significam

Durante a execução, você verá objetos destacados na tela do NX:

  • Curvas do perímetro externo — ficam destacadas (highlight) na cor padrão do NX
  • Linhas de centro de dobra — também destacadas visualmente

Os destaques são temporários e desaparecem quando o plugin termina.

📏 Cotas PMI geradas

Se você tiver a licença PMI/GD&T ativa, o plugin cria cotas PMI em cadeia que mostram as distâncias entre a borda externa e cada dobra: 

[borda externa] ← 29.85 → [dobra 1] ← 22.85 → [dobra 2] ← ...
Sem licença PMI? O plugin desenha linhas indicadoras vermelhas no lugar das cotas. Você ainda pode ver as distâncias no Log File.

As cotas recebem o atributo "FlatPatternHighlight" = "true", permitindo que você identifique quais cotas foram geradas pelo plugin.

5. Exemplos Práticos

📘 Exemplo 1: Peça simples com 2 dobras

Cenário: Uma chapa retangular com duas dobras simples (uma para cima, uma para baixo).

Procedimento:

  1. Abra a peça no NX
  2. Crie o flat pattern (Insert → Sheet Metal → Flat Pattern)
  3. Execute o plugin (Menu → Flat Pattern → Highlight Exterior Curves)
  4. Abra o Log File (Help → Log File)

O que você verá no log:

--- Outer Perimeter (True External Boundary) ---
  Outer perimeter curves: 4                      ← 4 segmentos = retângulo simples

--- Bend Center Lines ---
  Total bend lines: 2 (1 up, 1 down)             ← 2 dobras

--- Bend Line → Nearest Perimeter (Parallel) ---
  Bend[0] (UP)  → nearest: 15.2 mm              ← Dobra 1 está a 15.2 mm da borda
  Bend[1] (DOWN) → nearest: 22.8 mm              ← Dobra 2 está a 22.8 mm da borda

  PMI dimensions created: 2                      ← 2 cotas PMI criadas

📗 Exemplo 2: Peça complexa com recortes

Cenário: Uma peça com vários furos, notches e recortes internos — 74 curvas externas brutas.

O que o plugin faz de especial:

  1. Das 74 curvas, identifica que apenas 22 pertencem ao contorno externo verdadeiro
  2. As 52 curvas restantes (furos, notches) são ignoradas na análise
  3. Para cada dobra, o algoritmo detecta automaticamente se a curva mais próxima é um recorte fino (cutout) e a ignora, usando a segunda curva mais próxima como referência

Log característico:

  Outer perimeter curves: 22
  Inner (excluded): 52

  Bend[0] Dir=(1.000,0.000)
    Side A: nearest=   2.85  (cutout fino ignorado)
            2ndNearest= 22.85 (borda real usada)
            farthest  =380.00
✅ O plugin automaticamente detecta que o valor 2.85 mm é de um recorte e usa o 22.85 mm (a segunda curva mais próxima) como referência correta.

🔴 Exemplo 3: Diagnóstico de problemas

Situação: Você executa o plugin e recebe a mensagem:

No Flat Pattern feature found.

Causa provável: A peça ativa não possui um flat pattern. Nem toda peça de Sheet Metal tem um flat pattern — ele precisa ser criado explicitamente.

Solução:

  1. Vá em Insert → Sheet Metal → Flat Pattern
  2. Selecione a face de referência e configure os parâmetros
  3. Clique em OK para gerar o flat pattern
  4. Execute o plugin novamente

Situação: O plugin executa mas nenhuma cota PMI é criada:

PMI dimensions created: 0

Causas possíveis:

  • Sem licença PMI: O log mostrará No PMI license (948802). Neste caso, o plugin cria linhas vermelhas indicadoras — você ainda pode ver as distâncias no Log File.
  • Dobras diagonais sem candidato paralelo: O log mostrará No boundary curve found. O plugin usa o fallback de curva mais próxima.
  • Flat solid body não encontrado: O log mostrará Could not locate flat solid body. O plugin usa todas as curvas como fallback.

6. Dicas & Boas Práticas

Sempre gere o flat pattern antes

O plugin analisa o flat pattern existente. Certifique-se de que ele foi gerado (Insert → Sheet Metal → Flat Pattern) antes de executar o plugin.

Consulte o Log File primeiro

Antes de olhar para as cotas na tela, leia o Log File. Ele contém todas as distâncias de forma organizada e é mais rápido de consultar.

Use o log para documentação

Copie o conteúdo do Log File para um relatório ou planilha. Ele pode servir como documentação da análise da peça.

Dobras diagonais precisam de atenção

Se sua peça tem dobras em ângulo (não apenas horizontais/verticais), verifique no log se todas as dobras diagonais receberam cota. O plugin tem um fallback especial para elas.

Atualize o plugin regularmente

Verifique se há versões mais recentes do plugin. Cada versão traz melhorias nos algoritmos de análise e correção de casos especiais.

Teste em uma cópia da peça

O plugin cria cotas PMI e linhas indicadoras na peça. Se você não quiser alterar o modelo original, faça uma cópia antes de executar o plugin.

7. Perguntas Frequentes (FAQ)

❓ O plugin funciona no NX 2406 ou versões anteriores?

O plugin foi desenvolvido e testado no NX 2512. Versões anteriores podem funcionar, mas não são oficialmente suportadas. É necessário recompilar a DLL com as referências corretas para a sua versão.

❓ Preciso instalar algo além dos 3 arquivos?

Não. Apenas os 3 arquivos (.dll, .men, .rtb) na pasta startup são necessários. O NX carrega o plugin automaticamente.

❓ O plugin modifica o modelo 3D?

O plugin não modifica a geometria 3D da peça. Ele apenas:

  • Destaca objetos temporariamente (os highlights desaparecem ao final)
  • Cria cotas PMI (que podem ser apagadas manualmente se desejado)
  • Cria linhas indicadoras vermelhas (que também podem ser apagadas)
  • Escreve no Log File (que é apenas um registro de texto)

Todas as alterações no modelo se limitam a anotações PMI, que são totalmente removíveis.

❓ Como removo as cotas PMI criadas pelo plugin?

As cotas e linhas indicadoras criadas pelo plugin recebem o atributo "FlatPatternHighlight" = "true". Você pode identificá-las por este atributo e apagá-las manualmente no NX.

❓ O que significa "Flat solid body not found" no log?

Significa que o plugin não conseguiu localizar o corpo sólido do flat pattern dentro do NX. Isso pode acontecer em peças muito complexas ou com configurações especiais. O plugin então usa todas as curvas externas (incluindo recortes) como fallback — os resultados ainda são úteis, mas menos precisos.

❓ Quantas dobras o plugin consegue analisar?

Não há limite. O plugin já foi testado com peças contendo de 2 a 30+ dobras. O tempo de execução cresce linearmente com o número de dobras.

❓ O plugin funciona com peças em português?

Sim. O NX é agnóstico ao idioma para as operações de API que o plugin utiliza. O log é gerado em português ou inglês dependendo da configuração do NX.

❓ Como desinstalo o plugin?

Basta remover os 3 arquivos da pasta startup:

  1. Feche o NX
  2. Delete FlatPatternHighlight.dll, FlatPatternHighlight.men e FlatPatternHighlight.rtb da pasta startup
  3. Reinicie o NX — o plugin não estará mais disponível
